# ShopkeepReports

Gem for downloading shopkeep reports via mechanize

## Installation

Add this line to your application's Gemfile:

    gem 'shopkeep_reports'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install shopkeep_reports
